# Manuals and Specifications 📚

Ye page ek tutorial ki tarah hai — is repo ka kaam hai aapko JavaScript language ke basics ko cover karna. But jaise hi aap JavaScript ke basics samajh jaoge, to aapko **in-depth** samajhne ke liye ya **up-to-date** rehne ke liye kuch resources ki zarurat hogi, jinke baare mein aaj hum baat karenge.

---

## Specification

> **ECMA-262 specification** JavaScript ko formally define karta hai — isme language ke baare mein sabse zyada in-depth, detailed aur formalized information hoti hai. Ye sabse trustworthy source hai language ke details ke liye.
>
> Link: <https://www.ecma-international.org/publications/standards/Ecma-262.htm>

Yahan aapko JavaScript ki depth knowledge milegi, but starting mein ye page thoda hard lag sakta hai — kyunki ye itna formal hai ki roz roz use karne ke liye practical nahi hai. Jab kisi specific feature ka exact definition chahiye ho tabhi yahan aao.

### Latest Draft Specification

Naya specification version har saal release hota hai. Releases ke beech mein, **latest draft** yahan milega:

<https://tc39.es/ecma262/>

### Upcoming / Bleeding-Edge Features (TC39 Proposals)

Nayi features — jinmein woh bhi shamil hain jo abhi "almost standard" hain (yaani **Stage 3** proposals) — yahan milegi:

<https://github.com/tc39/proposals>

> **Note:** Ye proposals abhi finalized nahi hain — ye future mein JavaScript mein aane wali features hain, current standard ka hissa nahi hain. Isliye inhe production code mein use karte waqt dhyan rakho.

## Manuals

- **MDN (Mozilla) JavaScript Reference** — ye ek detailed reference manual hai jisme examples aur in-depth information hoti hai individual language functions, methods aur features ke baare mein. Ye beginners ke liye bhi useful hai, but iska main purpose ek *reference* hona hai, na ki step-by-step tutorial.

  Link: <https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference>

> **Tip:** Aksar MDN directly open karne se behtar hai ki aap internet search mein `"MDN [term]"` type karo. Jaise agar `parseInt` function ke baare mein padhna ho to search karo: `MDN parseInt`

## Compatibility Tables

JavaScript ek evolving language hai — naye features regularly add hote rehte hain. Agar dekhna ho ki koi feature kis browser ya engine mein support karta hai, to ye resources use karo:

| Resource | Kya kaam aata hai? |
|---|---|
| [caniuse.com](https://caniuse.com) | Per-feature support tables for browsers — jaise modern cryptography functions kaun support karta hai |
| [kangax.github.io/compat-table](https://kangax.github.io/compat-table) | Language features aur unhe support karne wale engines ki table |

Ye sab resources real-life development mein bohot kaam aate hain — language details aur unki support ke baare mein valuable information dete hain. Inhe yaad rakho (ya is page ko bookmark karo) jab bhi kisi specific feature ke baare mein in-depth jaankari chahiye.
